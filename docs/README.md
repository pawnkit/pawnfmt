# pawnfmt docs

These notes cover the parts of `pawnfmt` people usually need once they move past the quick start: configuration, CLI behavior, ignores, and the project layout.

- [Configuration reference](configuration.md)
- [Project notes](project-notes.md)

For a ready-to-edit config file, run:

```sh
pawnfmt --init-config
```

That writes a commented `pawnfmt.toml` with every option set to its default value.
